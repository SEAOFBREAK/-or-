# 学生管理系统or教务系统
毕设题目：基于Qt的教务系统设计与实现，身份有学生、教师和管理员。

有亿点水的毕设，只有增删改查功能T_T，建议仅做参考。

平台用的是QtCreator，数据库用的MySQL数据库，education_system.sql为数据库文件。
